1. html 和 css

第3周培训我们开始了html的学习之路,了解了什么是Html

超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用于创建网页的标准标记语言。html的内容

接着第四周我们开始对页面的样式进行美化，开始学习CSS。不过在课上我们主要是学习了什么是id和class选择器。以及css的三种选择方式。

CSS 是层叠样式表 ( Cascading Style Sheets ) 的简称,它是一种标记语言，属于浏览器解释型语言，可以直接由浏览器执行，不需要编译。 CSS语法由三部分构成：选择器、属性和值： selector {property: value} CSS的内容

第五周我们讲了html5一些常用的标签,主要是vedio,audio,table,ul,ol,form以及一系列不同type属性的input输入框标签

Html5是Html最新的修订版本.Html5的教程

    2. JavaScript

    JavaScript是一种脚本语言（弱类型）。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言， 最早是在HTML网页上使用，用来给HTML网页增加动态功能。

    • 基本语法
    • 事件
    • DOM
    • 对象
    学好js对现在来说相当重要，现在JavaScript的发展已经相当迅速。 Js(JavaScript的简称)配合上Linux内核已经可以做一个操作系统了，不过这不在我们的培训内容当中。 Js讲完之后可能会学nodeJs，它是Js周多框架的一个
    并且小程序的逻辑层也完全靠js处理. JavaScript的教程

    3. JQuery

    jQuery 教程 jQuery 是一个 JavaScript 库。它极大地简化了 JavaScript 编程。

    • 常用选择器(ID,元素和类名)
    • jQuery实现 动画效果 (即网站中的jQuery效果一章)
    • jQuery控制页面和事件
    学好JQuery, 有利于提高我们的开发效率,作为一个很流行的框架，希望大家能对它有所掌握。 建议扫一遍 jQuery教程(菜鸟教程) 或 jQuery教程(w3school)
    感兴趣可以看看 慕课网的这些视频 (挑喜欢的看一下挺好的)

    4.Nodejs

    Node.js是一个基于ChromeV8引擎的JavaScript运行环境。 node.js可以解析、执行js代码。

    • 回调函数
    • 监听事件
    • 使用fs模块读写文件
    • 搭建简单服务器
    学好nodejs，有利于我们使用JavaScript去进行后端开发。当时上课时我们没有提到 Express,这里建议大家看看， 毕竟这是一个很好的web框架,使用它可以提高我们的开发效率。学好nodejs也有利于我们使用js 进行全栈开发.可以看看慕课网的  nodejs全栈入门视频

    homework2(尽量每个人都要做)

    我们已经学习了html,css,js,jquery,nodejs, 这里可以给两个练习。 大家可以自由发挥。不求完成程度，只重学习过程。

    1. 写一个网页总结自己之前学的知识（亦可加其他元素），可以知识点写在表格，列表等里面， 也可以自己用其他容器配合起来，用上css让页面变得好看。
    2. 写个表单，实现注册页面（越详细越好），要求用js进行判断，比如是否为空，是否符合逻辑 （比如电话号码位数，并且一般以哪些数字开头等，这个js实现的功能不强求, 能实现啥就啥, 结果不重要，重要在学习的过程），然后尽量用上css，让它变得美观些。
