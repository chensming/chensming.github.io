html 和 css

第3周培训我们开始了html的学习之路,了解了什么是Html

超文本标记语言（英语：HyperText Markup Language，简称：HTML）是一种用于创建网页的标准标记语言。html的内容

接着第四周我们开始对页面的样式进行美化，开始学习CSS。不过在课上我们主要是学习了什么是id和class选择器。以及css的三种选择方式。

CSS 是层叠样式表 ( Cascading Style Sheets ) 的简称,它是一种标记语言，属于浏览器解释型语言，可以直接由浏览器执行，不需要编译。 CSS语法由三部分构成：选择器、属性和值： selector {property: value} CSS的内容

第五周我们讲了html5一些常用的标签,主要是vedio,audio,table,ul,ol,form以及一系列不同type属性的input输入框标签

Html5是Html最新的修订版本.Html5的教程

JavaScript

JavaScript是一种脚本语言（弱类型）。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言， 最早是在HTML网页上使用，用来给HTML网页增加动态功能。

  • 基本语法
  • 事件
  • DOM
  • 对象
学好js对现在来说相当重要，现在JavaScript的发展已经相当迅速。 Js(JavaScript的简称)配合上Linux内核已经可以做一个操作系统了，不过这不在我们的培训内容当中。 Js讲完之后可能会学nodeJs，它是Js周多框架的一个
并且小程序的逻辑层也完全靠js处理. JavaScript的教程
